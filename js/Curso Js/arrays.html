Arrays en Js

Propiedades de los arrays

  1. Contienen indices numéricos.
  2. Longitud dinámica.
  3. Tienen heterogeneidad.

Funciones en los Array de Js

  • Ver número de elementos: .length().
  • Último elemento: var[var.length -1].
  • Cambiar de "int" a "string": .toString().
  • Unir tipos de arrays: var.concat(var2).
  • Borrar el último elemento: .pop.
  • Agregar como último elemento: .push().
  • Eliminar el primer elemento: .shift().
  • Insertar como primer elemento: .unshift().
  • Eliminar a partir de un punto: .splice().
  • Copiar un array: .slice().
  • Ordenar alfabeticamente: .sort().
  • Orden alfabetico inverso: .reverse().

Se puden crear arrays cuyos valores no se puedan modificar, para definirlos usaremos const en su declaración

const nom_array = [valores];
